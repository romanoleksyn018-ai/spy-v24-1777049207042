Інструкція
1. Створи GitHub репо (може бути порожнім)
2. Створи Personal Access Token зі scope: repo + workflow
3. Введи токен і назву репо (формат: username/reponame)
4. Вставте код і натисни COMPILE
5. Компілятор автоматично:
   • Завантажить User_Setup.h та .ino у репо
   • Запустить GitHub Actions workflow
   • Слідкує за прогресом в реальному часі
   • Дає посилання на .bin для скачування

⚠ Час компіляції: ~3–6 хвилин (GitHub Actions встановлює toolchain)
